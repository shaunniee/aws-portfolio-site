Static Portfolio Site v2 (this site)

S3 • CloudFront • WAF • Route 53 • ACM • CodePipeline • CodeBuild

Static HTML/CSS portfolio hosted in a private S3 bucket, fronted by CloudFront with AWS WAF, custom domain and TLS from ACM. Deployed automatically from GitHub on every push to main.

  • Private S3 origin accessed only via CloudFront Origin Access Control (OAC).
  • WAF Web ACL with AWS managed rules and basic rate limiting in front of CloudFront.
  • CloudFront & WAF logs shipped to dedicated S3 buckets for analytics.
  • CodePipeline & CodeBuild sync content to S3 and invalidate CloudFront caches.

Classic 3-tier Task Manager

VPC • ALB • EC2 ASG • RDS • SSM • CloudWatch

A traditional 3-tier task management app to demonstrate VPC networking, routing, and operations. Built the “old school” way to contrast with the serverless stack.

  • Public subnets for ALB and NAT gateway, private subnets for app and DB tiers.

How this site is deployed

High-level summary

Code lives in a GitHub repo. A CodePipeline in eu-west-1 listens for pushes to main and triggers a CodeBuild project.

CodeBuild uses a buildspec.yml file to build and sync artifacts to a private S3 bucket, and then creates a CloudFront invalidation so new content goes live quickly worldwide.

CloudFront sits in front of S3 with an Origin Access Control, AWS WAF Web ACL, and an ACM certificate attached to the custom domain.
